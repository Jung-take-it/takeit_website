18장. 공개 API 활용

1. 공개 API란?

특정한 인증 없이 누구나 사용할 수 있도록 공개된 API입니다. 일반적으로 데이터 제공 목적이며, JSON 형식으로 응답합니다.

2. 대표적인 공개 API 예시

API 이름 제공 데이터 사용 URL 예시
JSONPlaceholder 가상의 게시글, 댓글, 사용자 https://jsonplaceholder.typicode.com/posts
OpenWeather 실시간 날씨 정보 https://api.openweathermap.org
Cat Facts 고양이 관련 잡학 https://catfact.ninja/fact

3. fetch를 이용한 공개 API 사용 예제

fetch('https://jsonplaceholder.typicode.com/posts/1')
  .then(res => res.json())
  .then(data => {
    console.log("제목:", data.title);
    console.log("본문:", data.body);
  })
  .catch(error => console.error("에러:", error));

4. 실시간 고양이 정보 불러오기 예제

<button onclick="getCatFact()">고양이 사실 보기</button>
<p id="fact"></p>

<script>
  function getCatFact() {
    fetch("https://catfact.ninja/fact")
      .then(res => res.json())
      .then(data => {
        document.getElementById("fact").innerText = data.fact;
      });
  }
</script>

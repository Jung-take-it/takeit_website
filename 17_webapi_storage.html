17장. 웹 API, 스토리지 API

1. 웹 API란?

웹 브라우저가 제공하는 기능으로, 자바스크립트에서 사용할 수 있는 내장 API입니다. 예: DOM API, Fetch API, Storage API 등

2. 스토리지 API

웹 페이지에 데이터를 저장할 수 있는 API입니다. 크게 두 가지 종류가 있습니다:

종류 특징 유지기간 용도
localStorage 영구 저장 브라우저 종료 후에도 유지 사용자 설정, 토큰 저장 등
sessionStorage 세션 저장 브라우저 탭 닫으면 삭제 임시 상태 유지

3. localStorage 사용 예시

// 저장
localStorage.setItem("username", "홍길동");

// 불러오기
const name = localStorage.getItem("username");
console.log(name); // "홍길동"

// 삭제
localStorage.removeItem("username");

4. sessionStorage 사용 예시

// 저장
sessionStorage.setItem("tempData", "임시 정보");

// 불러오기
console.log(sessionStorage.getItem("tempData")); // "임시 정보"

// 삭제
sessionStorage.removeItem("tempData");

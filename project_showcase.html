웹프로그래밍 프로젝트 문서

1. 프로젝트 개요

이 프로젝트는 GitHub Pages와 HTML, JavaScript를 활용하여 웹프로그래밍 기초 개념을 정리하고, 실제 동작 가능한 예제를 통해 개념을 학습하는 것을 목표로 합니다.

또한 Google Sheets, Google Forms를 통해 사용자 피드백과 실시간 통계 시각화를 구현하려 하였습니다.

3. 사용한 기술

  • HTML5 / CSS3
  • Vanilla JavaScript (ES6+)
  • fetch API / async-await
  • Google Forms, Google Sheets API 연동
  • GitHub Pages 배포

4. 어려웠던 점

- Google Forms에서 entry ID를 추출하여 JavaScript에서 자동 제출하는 부분이 까다로웠습니다.

- Google Sheets에서 실시간 통계를 JavaScript로 시각화할 때 `NaN` 오류가 자주 발생했습니다.

- GitHub Pages 반영이 바로 되지 않아 캐시 우회를 위한 URL 쿼리스트링 기법을 활용하였습니다.

5. 결과 및 개선사항

웹프로그래밍의 개념을 시각화와 함께 정리할 수 있었고, 비동기 통신 및 API 연동 실습 경험을 쌓을 수 있었습니다.

다만, 실시간 데이터 시각화는 완전히 구현하지 못해 향후 GAS(Google Apps Script)나 Chart.js 등과 연계해 개선할 계획입니다.

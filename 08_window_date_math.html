8장. window, date, math 객체

1. window 객체

브라우저 환경에서 최상위 객체이며, 전역 변수나 함수를 포함하고 있습니다. 팝업창, 타이머, 브라우저 정보 등 다양한 기능을 제어할 수 있습니다.

속성/메서드설명예시
window.alert()알림창을 띄움alert("Hello!")
window.confirm()확인/취소 창confirm("정말 삭제하시겠습니까?")
window.prompt()입력값 받기prompt("이름을 입력하세요")
window.setTimeout()지연 실행setTimeout(() => alert("1초 후"), 1000)
window.location현재 URL 정보console.log(location.href)

2. Date 객체

날짜와 시간을 다루는 내장 객체입니다. 현재 시간 출력, 특정 날짜 설정 등에 유용합니다.

메서드설명예시
new Date()현재 시간const now = new Date()
getFullYear()연도 가져오기now.getFullYear()
getMonth()월(0~11)now.getMonth()
getDate()일자now.getDate()
toLocaleString()로컬 문자열now.toLocaleString()

3. Math 객체

수학적 계산을 위한 내장 객체로, 정적 메서드로 사용됩니다.

메서드설명예시
Math.abs(x)절댓값Math.abs(-10) // 10
Math.round(x)반올림Math.round(4.7) // 5
Math.floor(x)내림Math.floor(4.9) // 4
Math.random()0~1 난수Math.random()
Math.pow(x, y)x의 y승Math.pow(2,3) // 8

예제

// 현재 날짜 출력
const now = new Date();
console.log(`오늘은 ${now.getFullYear()}년 ${now.getMonth()+1}월 ${now.getDate()}일입니다.`);

// 0~9 난수 출력
console.log(Math.floor(Math.random() * 10));

개인 프로젝트 상세

구현 페이지 소개

본 프로젝트에서는 사용자가 실제 강의 종료 시각을 공유하고 확인할 수 있는 웹페이지를 개발하였습니다. 메인 페이지(index.html)에서는 전체 기능에 대한 메뉴를 제공하며, 피드백 입력(feedback.html)과 통계 시각화(chart.html) 기능이 핵심 기능입니다.

주요 구현 원리

피드백 입력 페이지는 Google Sheets를 데이터베이스처럼 활용하여, 사용자가 입력한 교수명, 과목명, 종료시간 데이터를 Apps Script API를 통해 실시간으로 저장합니다. 저장된 데이터는 chart.html 페이지에서 JavaScript와 Chart.js를 활용해 시각화됩니다. 교수별 종료시간 평균을 계산하고 막대그래프로 표시함으로써, 사용자에게 유용한 비교 정보를 제공합니다.

교과목 평균 강의시간 안내 - 최종 보고서

1. 프로젝트 개요

이 웹사이트는 학생들이 수강 신청 전에 각 교과목의 실제 강의 종료 시각을 확인할 수 있도록 하여, 시간 관리와 강의 선택에 도움을 주기 위해 개발되었습니다.

2. 주요 기능 요약

  • 교수명, 과목명, 요일별 평균 강의 종료 시각 확인
  • 사용자 피드백을 통한 데이터 실시간 갱신
  • Chart.js 기반의 종료 시각 시각화

3. 시스템 구성도

아래는 Google Sheets를 활용한 시스템 구성도입니다.

4. 주요 구현 화면

① 메인 페이지

② 피드백 입력 화면

③ 시각화 결과 화면

5. 소감 및 향후 개선 방향

웹 기술과 구글 시트를 연동하여 실시간 피드백 반영이 가능한 시스템을 구현할 수 있어 매우 유익한 경험이었습니다. 추후에는 로그인 기반 피드백 제한, 모바일 반응형 개선, 시각화 기능 고도화를 추가하고자 합니다.
